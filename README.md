# Java学习资源库

这是一个Java学习的资源库，包含以下内容：

## 目录结构

- `英语.md` - Java相关英语词汇表，包含常用关键字、类名、方法名等专业术语的中英对照
- `docs/` - 学习文档和笔记
- `examples/` - 示例代码
- `exercises/` - 练习题

## 使用说明

1. 克隆仓库到本地
```bash
git clone [repository URL]
```

2. 浏览学习资源
- 查看英语词汇表以熟悉Java相关术语
- 阅读docs目录下的学习文档
- 运行examples目录下的示例代码
- 完成exercises目录下的练习题
